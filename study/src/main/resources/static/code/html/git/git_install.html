Mac安装Git

新版的Mac Pro不需要安装Git。

Linux安装Git

Linux各版本之间安装Git方式有差异，以新一点的Ubuntu为例 

$ sudo apt-get install git

Windows安装Git

搜索引擎搜索Git for Windows，或者点击https://git-for-windows.github.io/，下载安装。

注：Windows可能还需安装OpenSSH for Windows，搜索引擎搜索OpenSSH for Windows，或者点击https://sourceforge.net/projects/sshwindows/下载安装。
