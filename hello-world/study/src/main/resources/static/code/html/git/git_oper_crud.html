创建版本库

当前目录变为本地仓库 

$ git init
关联远程仓库 
$ git remote add origin git@github.com:17768646814/hello-world.git
添加文件至本地仓库 
$ git add -u
提交文件至本地仓库 
$ git commit -m "your commit reason"
查看git状态 
$ git status

克隆远程仓库至本地 

$ git clone git@github.com:17768646814/hello-world.git
本地仓库推送至远程仓库 
$ git push -u origin master
